This page lists the small server-side tools run by the Iris System, some of which are used by the client-side tools on irys.cc.

PluralKit avatar proxy ¶

  • Members: https://tools.irys.cc/pkavi/m/<id>
  • Systems: https://tools.irys.cc/pkavi/s/<id>
  • Groups: https://tools.irys.cc/pkavi/g/<id>

All <id> fields can be the 5-character ID; the UUID (as returned by the API); or, in the case of systems, a Discord account ID.

Falls back to the Discord "blurple" avatar if there was an error retrieving the requested avatar; or if the requested object does not have an avatar/icon set.

A file extension can optionally be appended to the URL, however, it is ignored - the returned image is always a JPEG.
